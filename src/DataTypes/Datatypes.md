## String
Strings can be declared in quotes, double quotes and backticks

```
 const str = "Sample String";
```

```
const str = `Sample
    String 
    in multiple lines
    `;
```
## Boolean
Can either be true or false 

``` 
    const a = 20;
    const b = 20;

    Boolean(a+b) -> true
    Boolean('true') -> true
    Boolean([]) -> true
    Boolean({}) -> true

    Boolean('') -> false
    Boolean() -> false
    Boolean(undefined) -> false
    Boolean(null) -> false
```
